# clover-games

A collection of educational minigames for kids.

## Games

| Game | Subject | Tech |
|---|---|---|
| [Space Phonics](space-phonics/) | Letters & phonics (Letters and Sounds progression) | PixiJS v8, no build step |

Each game is a self-contained static site. Serve the repo (or the game's
folder) over HTTP and open it in a browser:

```bash
python3 -m http.server 8000
# open http://localhost:8000/space-phonics/
```
