# Custom phonic sounds - drop your recordings here

Generate each sound in the ElevenLabs website (or record it any other
way), download the audio, rename it to match the table below, and upload
it to this folder (GitHub: **Add file -> Upload files**). The import
workflow trims, normalises and installs it into the game automatically,
then redeploys the site. A file here always wins over every generated
version; delete it to fall back.

Accepted formats: mp3, wav, ogg, m4a, webm, flac.
Name files by the **letter(s)** as shown - e.g. `p.mp3`, `sh.mp3`, `ai.mp3`.

## What to generate

Type the "say this" text into ElevenLabs speech generation, audition,
and regenerate until it sounds like a friendly teacher making the pure
sound (avoid letter names - it's "sss", never "ess").

Tips for ElevenLabs:
- Short prompts like `sss!` or `p!` usually give a clean single sound;
  if it spells the letter out, try wrapping it in a phrase and clipping,
  or write it phonetically ("puh" said quickly).
- You don't need every sound at once - upload the ones you have and the
  rest keep their current audio.
- The importer trims silence and levels the volume, so don't worry about
  gaps or loudness.
